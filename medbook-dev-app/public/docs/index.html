    Introduction

    This documentation aims to provide all the information you need to work with our API.

    As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

    Base URL

    http://localhost

    Authenticating requests

    This API is not authenticated.

    Authentication

    Authentication endpoints

    Login

    Gain access to the application

    Example request:

    curl -X POST \
    "http://localhost/api/login" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"email":"rerum","password":"id"}'

    const url = new URL(
    "http://localhost/api/login"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "email": "rerum",
    "password": "id"
}

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

    Example response (200):

    {
    "type": "authorization",
    "id": "token",
    "attributes": {
        "access_token": null,
        "token_type": null,
        "expires_in": null
    },
    "links": {
        "self": "http:\/\/localhost\/api\/login"
    }
}
    Received response:
    Request failed with error:

    Request      

    POST api/login

    Body Parameters

    email  string  
    The registered user email.

    password  string  
    The user secret code.

    Change password

    Set a new password that will be used to gain access the application

    Example request:

    curl -X POST \
    "http://localhost/api/change_password/1" \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -d '{"password":"placeat","password_confirmation":"asperiores"}'

    const url = new URL(
    "http://localhost/api/change_password/1"
);

let headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "password": "placeat",
    "password_confirmation": "asperiores"
}

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

    Example response (200):

    {
    "type": "authorization",
    "id": "logout",
    "attributes": {

    Request      

    POST api/change_password/{$link}

    URL Parameters

    $link  string  

    Body Parameters

    password  string  
    The new password that is a minimum of 8 characters.

    password_confirmation  string  
    Re-type the new password as a confirmation.
